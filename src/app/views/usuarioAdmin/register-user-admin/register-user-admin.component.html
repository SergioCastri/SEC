Registrar administradores

Ingresa a continuación los campos requeridos y presiona el botón "Enviar", recuerda tener a la mano la cuenta de correo electrónico del usuario que vas a ingresar, allí será enviada una contraseña asignada por el sistema para que este ingrese. Una vez sea enviada, el sistema solicitará que la cambie, una vez finalizado este proceso esta persona podrá hacer uso de nuestros servicios.
